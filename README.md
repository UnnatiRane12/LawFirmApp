# ⚖️ LawFirmApp

A modern Law Firm Management System designed to simplify legal operations by connecting **Clients, Lawyers, and Administrators** on a single platform. 
The application streamlines case management, appointments, communication, and document handling for a more efficient legal workflow.

## ✨ Features

### 👤 Client
- View and track case progress
- Book and manage appointments
- Securely chat with assigned lawyers
- Upload and access legal documents
- View case history and updates

### ⚖️ Lawyer
- Manage cases and client records
- Update case status and legal notes
- Communicate with clients through chat
- Manage appointments and schedules
- Upload and share legal documents

### 🛠️ Admin
- Manage lawyers and clients
- Monitor all cases and appointments
- Control user roles and permissions
- Oversee platform operations
- Access system-wide analytics

## 🚀 Tech Stack

- **Frontend:** Flutter
- **Backend:** Supabase
- **Database:** PostgreSQL
- **Authentication:** Supabase Auth
  

## 🎯 Objectives

- Digitize law firm operations
- Improve client-lawyer communication
- Simplify case and appointment management
- Reduce manual paperwork
- Provide a secure and centralized legal platform
